AUchievement
============

DevDays project

Инструкция по сборке на локальной машине:

Создать базу данных: для этого зайти в (project_dir), сделать 
```bash
Web/manage.py syncdb
```

Затем заполнить ее начальными значениями, для этого сказать
```bash
python initial_fill.py
```

Затем запустить главный скрипт, который качает документы, парсит их, обновляет базу данных (можно также запустить его с параметром -- см. вики, чтобы была постоянная подкачка данных)

```bash
python main.py
```

И запустить сервер:

```bash
Web/manage.py runserver
```
